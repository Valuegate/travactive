🌍 **Travactiv – Frontend Documentation**

Smarter Travel Decisions, **Powered by AI**

Travactiv is an AI-powered travel guide and travel inspiration web application designed to assist:

🎓 Students traveling abroad for studies

✈️ Tourists and global travelers planning trips

🌎 Explorers looking for curated, data-driven travel insights

Travactiv simplifies global travel information, visa rules, study-abroad guidance, and destination inspiration into one intelligent platform.

📌 **Project Overview**

Travactiv is built to:

Provide real-time travel updates

Offer AI-powered recommendations

Guide students through study abroad planning

Inspire travelers with curated destination insights

Deliver clear, fact-checked travel information

The platform contains two major dashboards:

🎓 #Student Dashboard

🌍 Travelers Dashboard
